How to create a Microsoft Edge desktop shortcut

Updated: 08/31/2020 by Computer Hope

  • Shortcuts are a great way to quickly access programs, files, folders, and even web pages. While we already have a page on how to create desktop shortcuts, making a shortcut for Microsoft Edge is a bit more tricky. The following steps show you how to put a shortcut on your desktop that opens the Microsoft Edge Internet browser.

Steps to create a Microsoft Edge desktop shortcut

  1. Right-click an empty area on the Windows desktop and select New in the drop-down menu that appears.

  2. Under New, select the Shortcut option.

  3. In the Type the location of the item text field (A), enter the following text.

    %windir%\explorer.exe shell:Appsfolder\Microsoft.MicrosoftEdge_8wekyb3d8bbwe!MicrosoftEdge

  4. Click the Next button (B).

    Instead of typing in the string above, you can copy and paste it to save time.

    Choose a name for your new shortcut icon (A), then click the Finish button (B).

  5. Locate and right-click the new shortcut icon on your desktop and select Properties from the drop-down menu that appears.
